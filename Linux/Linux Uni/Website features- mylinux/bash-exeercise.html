Bash Scripting - Concepts & Exercises

1. Introduction to Variables

Bash scripting supports two types of variables:

  • Local Variables: Set and used within the script.
  • Environment Variables: Set by the system and available to all processes.

Example:

#!/bin/bash
STR="Hello World!"
echo $STR

2. File & Directory Management

Some useful file commands:

  • ls - List directory contents
  • mkdir mydir - Create a directory
  • rm myfile.txt - Delete a file
  • mv oldname newname - Rename a file

3. Pipes & Redirection

Redirection allows output to be stored in files or passed between commands.

  • ls > files.txt - Save output to a file
  • cat file1.txt | wc -l - Count lines in a file

4. Grep & Regular Expressions

The grep command allows searching through files using patterns.

grep "error" logfile.txt

Exercises

Exercise 1: Create and Use a Variable

Write a script that stores your name in a variable and prints it.

Solution:

#!/bin/bash
NAME="Francis"
echo "Hello, my name is $NAME"

Exercise 2: Count Number of Files in a Directory

Use a command to count the number of files in the current directory.

Solution:

ls | wc -l

Exercise 3: Find Lines Starting with a Specific Letter

Search a file for lines that start with the letter 'A'.

Solution:

grep "^A" myfile.txt
